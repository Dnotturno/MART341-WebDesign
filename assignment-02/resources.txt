What is a directory, and what is its relationship to files?
  A directory is the path for organizing file systems and describes where files are located.
How are VCS useful?
  Version Control Systems are useful as they enable limitless collaboration on a file, allowing real-time changes to be reflected on the file. Saving also becomes easier as the VCS recognizes a file or set of files as only one version thus limiting confusion about file versions.
